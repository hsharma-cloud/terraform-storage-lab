# Terraform Azure Storage Deployment

This project demonstrates how to deploy Azure Storage infrastructure using Terraform.

The repository provisions a storage account and blob container in Microsoft Azure using Infrastructure as Code (IaC).

---

## Project Overview

This lab demonstrates:

- Infrastructure as Code using Terraform
- Azure Storage Account deployment
- Azure Blob container creation
- GitHub Actions CI workflow for Terraform validation

---

## Architecture

Terraform  
↓  
Azure Resource Group  
↓  
Azure Storage Account  
↓  
Azure Blob Container  

---

## Technologies Used

- Terraform
- Microsoft Azure
- Azure Storage (Blob)
- GitHub Actions
- Infrastructure as Code (IaC)
